You are a front-end developer.

This project is a website proposal for a logistics company called Around Europe B.V.

The main goal of the website is to present the company profile in a modern and professional way.
The company focus is shipping and logistics services around Europe.

For now, the required output is a stylish modern HTML template with row-based slides.
Each slide should be a full-width horizontal section with its own content and a clear visual identity.
The website must include a top main navigation.

Use modern HTML, CSS, and JavaScript.
The design must be responsive for desktop, tablet, and mobile screens.
Use `around_europe_logo.svg` as the primary company logo asset and brand reference.
The site must support three languages: Dutch, English, and German.

Design direction:
- Modern row slides instead of column slides
- Strong hero section
- Clear company profile presentation
- Clean logistics-focused visuals
- Blue and orange brand palette

Style tokens and design rules:
- Primary brand blue: `#123F7F`
- Secondary logistics blue: `#1C63BF`
- Primary accent orange: `#F68016`
- Deep accent orange: `#DB6500`
- Main text color: `#123F7F`
- Muted text color: `#5F6F85`
- Light background: `#F8F4EC`
- Secondary background: `#EEF4FB`
- Dark footer / dark section background: `#0C1E3C`
- Deep dark background: `#081426`
- Main border color: `rgba(18, 63, 127, 0.12)`
- Main surface color: `rgba(255, 255, 255, 0.72)`
- Strong surface color: `rgba(255, 255, 255, 0.9)`

Typography rules:
- Main heading font: `Space Grotesk`, sans-serif
- Accent / quote font: `Instrument Serif`, serif
- Use large bold headings with tight letter spacing for section titles and hero text
- Use clean sans-serif body text for readability across desktop and mobile
- Keep heading style modern, sharp, and structured

Layout rules:
- Use full-width slides as the main website structure
- Keep internal content aligned inside a readable central width
- Use strong spacing, rounded cards, and soft shadows
- Use a dark footer and dark support sections for visual contrast
- Keep Dutch as the default language, with English and German available as secondary languages
